# learning PHP
Aprende a programar PHP.

Con la IA, he recopilado todo lo que se puede aprender de este fantástico lenguaje. Guardado por si a alguien le sirve.



🧱 Bloque 1: Los Cimientos y el Entorno (Donde estamos hoy)

    Qué es un lenguaje de servidor (Backend).

    Configuración del entorno local (Tu laboratorio de pruebas).

    Sintaxis obligatoria, etiquetas y tu primer script.

    Las variables y cómo PHP maneja la memoria.

🧠 Bloque 2: Lógica y Toma de Decisiones

    Operadores matemáticos y de comparación.

    Estructuras condicionales (if, else, elseif, switch).

    Mutación de tipos (El casting que vimos antes, pero aplicado a la lógica).

🔄 Bloque 3: Estructuras de Datos y Bucles

    Arrays indexados y asociativos (Organizar datos como un profesional).

    Bucles para repetir tareas (for, while, foreach).

📦 Bloque 4: Modularidad y Buenas Prácticas

    Creación de funciones personalizadas.

    Ámbito de las variables (locales vs. globales).

    Inclusión de archivos (include y require).

🌐 Bloque 5: PHP en el Mundo Real (La Web Dinámica)

    Cómo viajan los datos: Métodos GET y POST.

    Procesamiento de formularios y validación de seguridad.

    Mantener al usuario conectado: Sesiones ($_SESSION) y Cookies.

🗄️ Bloque 6: Persistencia (Bases de Datos)

    Introducción a las bases de datos (MySQL/MariaDB).

    Conexión segura mediante PDO.

    Operaciones CRUD (Crear, Leer, Actualizar y Borrar registros).
